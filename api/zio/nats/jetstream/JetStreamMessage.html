JetStreamMessage

zio.nats.jetstream.JetStreamMessage
See theJetStreamMessage companion object
final class JetStreamMessage(val subject: Subject, val replyTo: Option[Subject], val headers: Headers, val payload: Chunk[Byte], val underlying: Message)

A NATS message from a JetStream consumer, augmented with acknowledgment operations.

Contains the same payload data as zio.nats.NatsMessage but also provides methods to ack/nak the message within the JetStream at-least-once delivery model.

Obtained from Consumer.fetch, Consumer.consume, Consumer.iterate, or Consumer.next.

Value parameters

headers

NATS headers attached to the message.

payload

Raw message payload bytes.

replyTo

Optional reply-to subject.

subject

The message subject.

underlying

The underlying jnats io.nats.client.Message (used for ack only).
