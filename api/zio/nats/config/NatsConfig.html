Configuration for a NATS connection.

Provides a Scala-friendly API over io.nats.client.Options.Builder.

=== Authentication ===

Set the auth field to one of the NatsAuth variants. Only one authentication method can be active at a time; impossible combinations (e.g. token and user/password simultaneously) cannot be constructed. Use NatsAuth.Custom for dynamic credential rotation via a programmatic io.nats.client.AuthHandler.

=== TLS ===

Set the tls field to one of the NatsTls variants. Use NatsTls.Custom for a pre-built javax.net.ssl.SSLContext when certificates are loaded at runtime (e.g. from a secrets manager).

Value parameters

auth

Authentication method (default: NatsAuth.NoAuth). Use NatsAuth.Token, NatsAuth.UserPassword, or NatsAuth.CredentialFile for authenticated connections.

bufferSize

Size of the outbound message buffer in bytes (default: 64 KiB).

connectionName

Optional client name shown in server logs and monitoring.

connectionTimeout

Maximum time to wait for the initial connection (default: 2s).

discardMessagesWhenOutgoingQueueFull

If true, new messages are discarded when the outbound queue is full instead of blocking (default: false).

drainTimeout

Maximum time to wait for subscriptions to drain when the connection's ZLayer scope ends (default: 30s).

inboxPrefix

Prefix for auto-generated reply-to inboxes (default: _INBOX.).

maxControlLine

Maximum length of a NATS protocol control line in bytes; must match the server's max_control_line setting (default: 0 = jnats default).

maxMessagesInOutgoingQueue

Maximum number of messages in the outbound queue (default: 0 = unlimited).

maxPingsOut

Maximum number of client PINGs in-flight before the connection is considered stale and closed (default: 2).

maxReconnects

Maximum reconnect attempts before giving up (default: 60; -1 = unlimited).

noEcho

If true, messages published by this connection are not echoed back to its own subscriptions (default: false).

pingInterval

Interval between client-initiated PING messages (default: 2m).

reconnectBufferSize

Maximum bytes to buffer for in-flight publishes during a reconnect window (default: 8 MiB). Set to 0 to disable buffering.

reconnectJitter

Random jitter added to reconnectWait to avoid thundering herd on reconnect (default: 100ms).

reconnectJitterTls

Jitter for TLS connections, which are more expensive to establish (default: 1s).

reconnectWait

Base delay between reconnect attempts (default: 2s).

requestCleanupInterval

How often stale request inboxes are swept (default: 5s).

servers

List of NATS server URLs (default: nats://localhost:4222).

socketReadTimeout

Socket-level read timeout to prevent hanging on a dead connection (default: Duration.Zero = no timeout).

socketWriteTimeout

Socket-level write timeout to prevent hanging on a dead connection (default: Duration.Zero = no timeout).

tls

TLS configuration (default: NatsTls.Disabled). Use NatsTls.SystemDefault for public-CA-signed servers, or NatsTls.KeyStore for mTLS.

utf8Support

Enable UTF-8 subject support (default: false).

writeQueuePushTimeout

How long to wait when the outbound queue is full before giving up (default: Duration.Zero = block indefinitely).
